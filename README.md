# Dotfiles

## Requirements

- Git
- Curl

## Restore script

Install config tracking in your \$HOME by running:

```
curl -Lks https://raw.githubusercontent.com/echounit1/dots/master/restore.sh | /bin/bash
```

## Git commands

`dots` acts as an alias for `git` in the `$HOME` directory.
