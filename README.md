# Dotfiles

## Requirements

- Git
- Curl

## Restore script

Install config tracking in your \$HOME by running:

```
curl -Lk https://gitlab.com/andres-dc/dots/-/raw/main/.restore | /bin/bash
```

## Git commands

`dots` acts as an alias for `git` in the `$HOME` directory.
